# Snake Game using Python

This project is a classic Snake Game developed using Python Turtle graphics as part of an academic project.

## Features
- Snake movement using keyboard controls
- Food generation
- Score and high score tracking
- Game over detection

## Technologies Used
- Python
- Turtle Graphics
- Tkinter

## How to Run
1. Install Python on your system.
2. Download or clone this repository.
3. Run the file `snake_game.py` on a local system (VS Code / IDLE).
   
> Note: Turtle graphics require a graphical user interface, so this project cannot run on online platforms like Google Colab.
